# SafeAnchor Documentation

This folder contains the product, architecture, and learning documentation for SafeAnchor.

## Product Documentation

- `product/mvp.md`: MVP definition, audience, value proposition, scope, and metrics.
- `product/roadmap.md`: Epic and feature roadmap for the SafeAnchor MVP.
- `product/business-rules.md`: Business rules by domain module.

## Architecture Documentation

- `architecture/frontend.md`: Target frontend architecture using Next.js, TypeScript, Redux Toolkit, and MVVM.
- `architecture/backend.md`: Target backend architecture using NestJS, PostgreSQL, Prisma ORM, and TypeScript.
- `architecture/database.md`: Entity model, relationships, constraints, and ERD.

## Learning Documentation

The `learning` folder keeps beginner-friendly notes and step-by-step study material. These files describe the current simplified lab structure and should remain separate from the target SaaS architecture.
