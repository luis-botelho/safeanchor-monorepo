# Roadmap de Aprendizado

## Fase 1: Rodar o Projeto

- [ ] Rodar `npm run install:all`.
- [ ] Rodar `npm run dev:backend`.
- [ ] Abrir `http://localhost:3001/modules`.
- [ ] Rodar `npm run dev:frontend`.
- [ ] Abrir `http://localhost:5173`.

## Fase 2: Entender o Frontend

- [ ] Ler `apps/frontend/src/views/App.jsx`.
- [ ] Ler `apps/frontend/src/viewmodels/useHomeViewModel.js`.
- [ ] Ler `apps/frontend/src/services/moduleService.js`.
- [ ] Ler `apps/frontend/src/styles/global.css`.
- [ ] Alterar um texto e ver na tela.

## Fase 3: Entender o Backend

- [ ] Ler `apps/backend/src/server.js`.
- [ ] Ler `apps/backend/src/routes/modulesRoutes.js`.
- [ ] Ler `apps/backend/src/controllers/modulesController.js`.
- [ ] Ler `apps/backend/src/services/modulesService.js`.
- [ ] Adicionar um modulo novo na lista.

## Fase 4: Praticar MVVM

- [ ] Criar componente `ModuleCard`.
- [ ] Criar um novo campo no model.
- [ ] Atualizar service do backend.
- [ ] Atualizar viewmodel se precisar.
- [ ] Mostrar o campo na view.
