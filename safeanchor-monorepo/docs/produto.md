# Produto

SafeAnchor sera, aos poucos, um app para organizar informacoes maritimas.

Por enquanto, o produto tem apenas tres modulos de estudo:

- Embarcacoes
- Manutencoes
- Checklists

## Regra do Laboratorio

Antes de criar uma feature grande, faca uma versao pequena com dados fixos.

Exemplo:

1. Primeiro mostre uma lista fixa.
2. Depois busque essa lista no backend.
3. Depois pense em formulario.
4. So depois pense em banco de dados.
